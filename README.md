# Secrets Folder

This folder contains sensitive files managed by git-secret.

## Important Notes

- **DO NOT** commit unencrypted sensitive files to git
- Only encrypted files (*.enc) are tracked by git
- Use `git-secret lock` before committing changes
- Use `git-secret unlock` after pulling changes

## Workflow

1. Add your sensitive files to this folder
2. Run `git-secret lock` to encrypt them
3. Commit and push the encrypted versions
4. On other machines, run `git-secret unlock` to decrypt
